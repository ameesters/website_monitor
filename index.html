Setup:

Make sure the pygtk module is installed, and a gtk libs of version 2.0 or higher.

First we need to execute some commands, so open a comandline and enter:

$ git clone git@github.com:ameesters/website_monitor.git
$ cd website_monitor/
$ chmod +x init_db.py main.py
$ sqlite3 site_monitor.db
$ ./init_db.py
$ ./main.py

if not, please use the issue tracker! PS) For people who get stuck in the sqlite shell try .exit ;-)
